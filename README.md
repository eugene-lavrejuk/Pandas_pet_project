# Melbourne Housing Data Analysis
Проект по анализу рынка недвижимости Мельбурна с использованием Python, Pandas, Seaborn и Matplotlib.
## Описание
Этот скрипт выполняет полный цикл анализа данных:
- Загрузка датасета  
- Первичный анализ (EDA)  
- Очистка и предобработка данных  
- Создание новых признаков  
- Аналитика и инсайты  
- Визуализация  
- Экспорт данных  
Источник данных:  
https://raw.githubusercontent.com/esabunor/MLWorkspace/master/melb_data.csv
---
## Используемые технологии
- Python 3  
- pandas  
- numpy  
- matplotlib  
- seaborn  
---
## Этапы анализа
### 1. Первичный анализ данных (EDA)
- Общая информация о датасете  
- Проверка пропусков  
- Описательная статистика  
- Анализ категориальных признаков  
---
### 2. Очистка данных
Что было сделано:
- Исправлены опечатки в названиях колонок:  
  - Lattitude → Latitude  
  - Longtitude → Longitude  
- Удалены ненужные колонки:  
  - Unnamed: 0, Address, Bedroom2  
- Преобразована дата:  
  - Добавлены Month и Year  
- Обработка пропусков:  
  - Landsize, BuildingArea → медиана по типу дома (Type)  
  - YearBuilt → общая медиана  
- Удалены строки с критическими пропусками:  
  - CouncilArea, Latitude, Longitude  
- Удалены выбросы:  
  - Rooms <= 10  
  - Price < 6,000,000  
- Исправлен тип данных:  
  - Car → int64  
  - Postcode → int64
  - Bathroom → int64
---
### 3. Feature Engineering (новые признаки)
Добавлены:
- Age — возраст дома  
- Price_per_sqm — цена за квадратный метр  
---
### 4. Аналитика
Получены ключевые инсайты:
- Дома (Type = h) — самые дорогие (около 1.3 млн AUD)  
- Определены топ-5 дорогих регионов  
- Найдены топ-10 риелторов по объёму продаж  
- Рассчитана цена за м² по типам недвижимости  
---
### 5. Корреляционный анализ
Факторы, влияющие на цену:
- Rooms — сильная положительная корреляция (~ +0.55)  
- BuildingArea — умеренная (~ +0.36)  
- Distance — отрицательная (~ -0.28)  
- Age — слабая отрицательная (~ -0.10)  
---
### 6. Визуализация
Строятся графики:
- Средняя цена по типу жилья  
- Топ регионов  
- Тренд цен по месяцам  
- Географическая карта цен  
Есть возможность сохранить графики в PNG (Melbourne_GRAPH.png)
